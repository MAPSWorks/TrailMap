
TiffSplit
---------

This program splits large TIFF files into many small tiles for easier handling of large maps.
Currently the only supported input filetype is 'Maastokarttarasteri 1:50 000'.
These files can be downloaded from https://tiedostopalvelu.maanmittauslaitos.fi/tp/kartta.


Building
--------

This project uses LibTIFF.Net. Obtain a binary copy from http://bitmiracle.com/libtiff.net_bin.zip
and unzip the contents to libtiff.net folder in the source tree.

Use Microsoft Visual C# 2008 Express Edition to build.
